1.下载Node.js 
下载链接：https://nodejs.org/en/download

2.项目
安装完成nodejs后，

第二步
点击页面的Add file 选择 Upload files，下载并解压文件 注意复制路径

WEIN+R打开CMD 命令行

项目根目录使用：
npm install 
等待安装依赖成功之后
在server.js中 填写私钥、RPC节点、上级地址等数据

项目目录使用：node server.js 即可运行

注：
运行后可使用 ctrl + c 中断运行
